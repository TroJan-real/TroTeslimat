# Troteslimat

Haftalık teslimat yarışı plugini. Oyuncular belirli bir eşyayı haftalık olarak teslim ederek sıralamada yükselir.

![Minecraft](https://img.shields.io/badge/Minecraft-1.21+-brightgreen)
![Java](https://img.shields.io/badge/Java-21-blue)
![License](https://img.shields.io/badge/License-GPLv3-red)

## Özellikler

- 🎨 Modern ve şık GUI arayüzü
- 📊 Canlı leaderboard + sayfalı Top Teslimat GUI'si
- 🌐 Discord entegrasyonu (`/teslimat` slash komutu + otomatik haftalık embed)
- ⚙️ Kolayca yapılandırılabilir config
- 🛠️ Güçlü admin komutları
- 📅 Otomatik haftalık sıfırlama sistemi

## Komutlar

| Komut              | Açıklama                          | Yetki                    |
|--------------------|-----------------------------------|--------------------------|
| `/teslimat`        | Teslimat menüsünü açar            | `troteslimat.use`        |
| `/topteslimat`     | Top teslimat sıralamasını açar    | `troteslimat.use`        |
| `/troteadmin`      | Admin komutları                   | `troteslimat.admin`      |

**Admin Alt Komutları:** `setitem`, `reset`, `top`, `reload`

## Kurulum

1. En son sürümü [Releases](releases) sekmesinden indirin.
2. `Troteslimat.jar` dosyasını `plugins` klasörüne atın.
3. Sunucuyu başlatın.
4. `plugins/Troteslimat/config.yml` dosyasını düzenleyin.
5. Sunucuyu yeniden başlatın.
